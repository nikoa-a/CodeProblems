## Counting Bits

Given an integer n, return an array ans of length n + 1 such that for each i (0 <= i <= n), ans[i] is the number of 1's in the binary representation of i.

---

#### Example 1:
> **Input:** n = 2<br>
> **Output:** [0,1,1]<br>
> **Explanation:**<br>
> 0 --> 0<br>
> 1 --> 1<br>
> 2 --> 10

#### Example 2:
> **Input:** n = 5<br>
> **Output:** [0,1,1,2,1,2]<br>
> **Explanation:**<br>
> 0 --> 0<br>
> 1 --> 1<br>
> 2 --> 10<br>
> 3 --> 11<br>
> 4 --> 100<br>
> 5 --> 101
